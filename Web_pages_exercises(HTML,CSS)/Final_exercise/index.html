Why is design thinking
important?
Design thinking enables
organizations to create lasting
value for consumers. The
process is useful in any
complex system (not just
design systems) because it:

• Aims to solve a concrete human
need

• Tackles problems that are
ambiguous or difficult to define

• Leads to more innovative solutions

• Makes organizations run faster
and more efficiently
